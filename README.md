# FS-Communication

A design-only tech products store and finance dashboard built with Next.js and TypeScript.

## Last Commit Message

**Initial project snapshot**

## Project Structure

- `src/app/` – Next.js App Router pages
- `src/components/` – Reusable UI components

## Tech Stack

- **TypeScript** – App structure
- **React / Next.js** – Pages and layout
- **Tailwind CSS** – Design and styling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
