# FS Communication Project Overview

This project is a simple tech products store design.

## Landing Page
- A clean homepage that showcases tech and electronics products.
- It is designed for browsing and shopping.
- Users can view products and prices, but they cannot place orders yet.

## Navbar
- The navbar includes a login option.
- Login is available for two roles: Admin and Staff.

## Admin
- The admin panel should include:
  - Dashboard with cash balance, bank balance, receivables, payables, customer advances, supplier advances, today's sales, today's expenses, net profit, daily profit, monthly profit, and today's recovery.
  - Charts for invoices over time and sales.
  - A recent invoices table.
  - A business overview summary.
  - Report buttons.
  - Navigation breadcrumbs such as Back, Home, and Dashboard.
  - Sections for Sales, Purchases, Payments, Parties, Reports, and Settings.
  - Settings where the admin can create staff accounts and assign access to specific features, such as view-only or edit permissions.

## Staff
- Staff access is controlled by the admin.
- Staff members can use only the features assigned to them.

## Languages and Design Stack
- TypeScript for the app structure.
- React and Next.js for the pages and layout.
- CSS for the design and styling.
- The current work is focused on design only.

## Current Status
- This project is design-only for now.
